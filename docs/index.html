class fredlib._core.Database(db_name: str)

Bases: fredlib._core.DataManager

This class is used to interact with an SQLite database. Its main purpose is to write data downloaded from FRED into a local database. It also allows to fetch the saved data from the database, parsing them into appropriate objects.

class fredlib._core.Fred(key)

Bases: fredlib._core.DataManager

This class is used to interact with FRED’s services. It makes HTTP requests in order to retrieve the desired data in json format. It also parses json responses, building the appropriate ModelType object representing the data.

_get(query)

This private method makes a query to FRED’s services; if the request does not fail, the method returns a json object

Parameters

query (str) – The initial URL part of the query to FRED’s services, without specifying the API key

Raises

BadRequestException – The method throws an exception if the request fails

Returns

The json object resulting from the query

Return type

Any

_parse(json_object, model_type, elem_id=0) → List[Any]

This method parses a json object into a list of objects whose type is one of those in ModelType. When parsing observables, only those that have no NaN value are kept.

Parameters

  • json_object (str) – The JSON document to be parsed

  • model_type (ModelType) – The type of expected resulting object

  • elem_id (int) – It represents the category [resp. series] identifier to which a Series [resp. Observable] belongs

Raises

NotSupportedModelType – The exception is raised when the model_type parameter is not one of the enum ModelType.

Returns

A list of the requested objects.

Return type

List[Any]
